Hexo建站基本教程

发表于 2018-12-10 | 更新于 2019-02-14

All the trees are losing their leaves,and not one of them is worried

前言

正如Hexo官网的大标题所述,Hexo是一款快速、简洁且高效的博客框架。它拥有插件,通过安装各种插件,用户可以快速定制属于自己的静态网站。通过安装不同的主题,也使得博客样式丰富多彩,可以说,通过Hexo制作的博客都是独一无二的。现在就让我来教大家,如何利用Hexo制作自己的个人博客网站吧~

晔清2018.12.09 Sunday

安装Hexo所需依赖

  • Node.js
  • Git

安装Node.js

可以直接到Node.js的官网下载10.14.1 LTS(长期支持版本)
一路默认安装即可
安装完成后在cmd或者git bash中,输入node --version检验node是否安装成功

安装Git

官方网址:Git

安装中的具体设置根据喜好选择即可,新手建议默认设置就好



注意:国内可能无法直接从官网下载,可以访问国内镜像下载

安装Hexo

安装完Git后,通过Win+S搜索Git Bash
在Git Bash中,通过npm install -g hexo-cli来获取最新版的Hexo

如果你不想花钱租用昂贵的服务器,那么你可以选择将项目部署到Coding上。

小结

此篇文章仅为Hexo官网教程的个人理解,选择Coding代码仓库托管代码,主要是由于Github服务器国内访问速度过慢。另外,Hexo的详细配置,我想额外开一篇文章,可能会放在Next主题配置下,一并讲解。
