# Vue-Cli 创建项目的步骤

  1. 首先使用yarn global add @vue/cli或者 npm install -g @vue/cli安装vue-cli3.0+
  2. 通过vue --version命令检查vue-cli是否安装成功
  3. 使用vue create 项目名创建一个vue-cli项目
  4. 选择项目的配置，一般不要选择default
  5. 我选择的是vue-router,css预处理器(less),babel
  6. 项目生成后，通过cd 项目名，yarn serve启动项目
    • public文件夹: 存放SPA单页面应用的入口html，以及图标
    • src文件夹: 存放项目的所有源文件
    • src
      • assets:存放项目的静态资源文件
      • components:存放项目的组件
      • views:存放项目的视图
      • App.vue:项目的根视图，整个页面的主体
      • main.js:主要用来引入全局插件和项目的Vue实例
      • router.js:项目的路由文件，所有路由都需要在这里注册

一个view视图可以引入多个组件，组件可以是一个搜索框，或者一个分页器

而视图是一个页面，比如新闻页面，推荐页面，首页。而这里的视图，是一个页面的路由。

# Vue-axios实现ajax请求

  1. 首先安装axios和vue-axios

    yarn add axios ,yarn add vue-axios

    yarn add axios vue-axios

  2. 在入口文件main.js中引入模块

    import axios from 'axios'

    import VueAxios from 'vue-axios'

    Vue.use(VueAxios,axios)

    注意必须得按顺序请求，请求必须在import Vue from 'vue'后，

    为什么要引入Vue,因为需要使用Vue.use()方法

  3. 使用方法:

    Vue.axios.get(api).then((response) => {
  console.log(response.data) //这个Vue.axios可能不能使用
})

this.axios.get(api).then((response) => {
  console.log(response.data)
})

this.$http.get(api).then((response) => {
  console.log(response.data)
})
