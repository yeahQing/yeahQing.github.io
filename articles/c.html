# Mysql8.0的安装及破解Navicat Premium12.1

# Mysql8.0的安装

WIndows平台下的安装:https://dev.mysql.com/downloads/mysql/

  1. 进入上述网址后，在下方找到

  1. 找到Windows ZIP Archive(Windows压缩文件)，第二个为带调试版本的Mysql，可以不用下载

  2. 我下载的版本为8.0.16

  3. 下载好之后解压到指定目录下。

  4. 在解压后的目录下，创建一个my.ini的配置文件，用来生成mysql8.0的基本配置

  5. my.ini的文件内容

    因为mysql8.0默认自动生成data目录，所以无需指定。且mysql8.0默认字符集为utf8mb4，所以此处为了统一，设置成utf8mb4(如果设为别的字符集)，在安装时可能会导致mysql报警告。

  6. 进入E:\xycreatejar\mysql-8.0.16-winx64\bin

  7. 在cmd中找到上述路径，执行mysqld --initialize --console初始化数据库

  8. 成功执行后，会看到cmd中有一行 root@localhost: [密码]，此处为用户名@端口号:初始密码

  9. 保存上述信息，以备登录使用

  10. 之后还在当前目录下执行 mysqld install。若安装成功，会出现一行successfully的安装字样

  11. 之后启动mysql服务

    • net start mysql

  12. 登录mysql

    • mysql -h 主机号 -u 用户名 -p
    • 此处会显示 Enter password: [输入保存的初始密码]
    • 如果登录成功，会出现mysql> 字样

  13. 修改用户初始密码

    • 使用 ALTER User User() IDENTIFIED BY '[密码]';
    • 修改成功后，使用exit推出数据库，使用新密码重新登录

# Navicat Premium的安装和破解

# 1.从官网下载最新版本的Navicat Premium

https://www.navicat.com.cn/

# 2.安装Navicat Premium

此处根据自己的喜好安装即可

# 3.找到破解补丁

此处使用大佬在github上的开源项目

https://github.com/DoubleLabyrinth/navicat-keygen

# 4.破解流程

  1. 下载最新的RELEASE发行版

  2. 将下载好的压缩包解压到任意目录(该压缩包无毒，若检测到病毒，请立即删除)

  3. 之后使用命令行工具，找到解压后的目录，执行如下指令

    navicat-patcher.exe <你Navicat的安装根目录>

  4. 当看到MESSAGE: Patch has been done successfully.这句话时，证明破解补丁已经写入了Navicat中

  5. 之后使用如下图所示方式生成序列号和激活码

    最新版本可以直接使用 navicat-keygen.exe -text .\RegPrivateKey.pem

  6. 之后选择你需要破解的软件，以及产品的产品的语言，软件的当前版本

  7. 保存生成的序列号 Serial number

  8. 输入自定义的姓名和组织

  9. 断网，打开Navicat,点击注册，输入保存的序列号，之后选择手动激活，将出现的请求码赋值到cmd中

  10. 按两下回车，复制生成的License,点击激活，即可完成激活
