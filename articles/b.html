# Maven学习笔记

# 1.Maven是什么？

Maven是一个项目包管理工具,它可以在中央仓库中托管jar包，大大减小项目的大小。

# 1.1 Maven中央仓库

https://search.maven.org/

https://mvnrepository.com/

# 2.Maven的安装

  • 从Maven官网http://maven.apache.org/index.html下载二进制压缩包

  • 解压缩到纯英文目录下

  • 配置环境变量

    • 首先新建配置名 %MAVEN_HOME%，配置路径为 安装目录
    • 之后配置path路径，%MAVEN_HOME%\bin
    • 注意要想使Maven正常运行，必须要有相应版本的jdk支持，即需要配置jdk的环境变量

# 3.Maven常用命令

# 3.1 mvn Clean命令

  • mvn clean:可以清除target目录

# 3.2 mvn Compile命令

  • mvn compile:可以生成target目录
  • 将项目中的.java编译成.class

# 3.3 mvn test 命令 单元测试

  • 单元测试类名有要求: 需要为xxxTest.java
  • 将项目根目录下 src/test/java目录的下的单元测试类都会执行

# 3.4 mvn package 打包

  • web project -- 打包成war包
  • java project -- 打包成jar包
  • 将项目打包，打包项目根目录下target目录

# 3.5 mvn install 安装

  • 将war/jar包提交到maven仓库
  • 本地仓库公用一个jar包

# 4. Eclipse配置Maven

  • 找到Preferences下的Mave->Installations

  • 找到Maven下的User Settings

  • 构建索引

    • nav栏中找到Windows，选择show views->other->搜索Maven Repositories

    • 找到Local Repositories->local repository

    • 右键Local Respository，选择rebuild index(重建索引)

# 5. 创建Maven项目

# 5.1 创建Maven Project

勾选Create a simple project可以跳过某种选择

  • Group Id: 公司名 用com/cn/org.公司名
  • Artifact Id: 项目名
  • Version: 版本号-测试版/发行版(SNAPSHOW/RELEASES)
  • Packaging: 打包方式(jar/pom/war)

# 6. Maven项目拆分

# 6.1 引入父工程

  • ssh_parent

    • ssh_web
    • ssh_service
    • ssh_dao

  • 创建父工程

    1. 创建一个maven项目maven project
    2. 在创建时选择打包方式Packaging 为pom

  • 创建子工程

    1. 创建一个maven项目maven module
    2. 创建时选择打包方式Dao和Service层选择为jar
    3. Web层选择为war
