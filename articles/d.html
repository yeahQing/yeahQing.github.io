    # CentOS下安装mysql 全量包

    到mysql官网找到RedHat

    https://dev.mysql.com/downloads/mysql/

    下载Linux7版本

    使用 tar 命令 tar -xf(x 为解压,f 为解压那个问价)

    tar -xf mysql-8.0.16-2.el7.x86_64.rpm-bundle.tar

    之后使用sudo rpm -ivh mysql-community-xxx-xxx.rpm --nodeps --force解压所有rpm压缩文件

    删除命令:rpm -e mariadb-libs-5.5.56-2.el7.x86_64 --nodeps

    rpm -qa | grep mysql 命令查看 mysql 的安装包

    mysqld --initialize : 这句话如果无效，则执行一下安装

    yum -y install numactl

    systemctl start mysqld.service

    如果在mysqld.log中没有生成临时密码

    • 需要删除/var/lib/mysql下的全部文件
    • 使用 rm -rf /var/lib/mysql

    无法使用 systemctl start mysqld.service

    原因是因为 /var/run/mysqld ,mysql没有该文件的访问权限

    利用chown命令改变该文件夹以及其下的文件的拥有者

    chown -R mysql:mysql /var/run/mysqld

    # 使用Nacivat远程连接服务器数据库

    首先在腾讯云中生成私钥

    保存好私钥，并在腾迅云实例中，加载密钥

    在Navicat中选择SSH登录

    主机: 服务器公网ip

    端口号: 22

    用户名: root

    验证方法: 公钥

    选择自己保存好的私钥文件，

    点击测试连接

    之后在常规里，连接localhost本地数据库，即首先连接到服务器，之后通过服务器连接服务器的本地数据库
